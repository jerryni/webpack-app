# webpack base

## feature

- babel
- eslint
- scss-loader, file-loader, url-loader, html-loader
- multiple entry

## Start 

> npm i

develop with live-server:

> npm run dev

build files to path `./build`
> npm run build

## file tree
```js
├── app
│   ├── index.js
│   └── scss
│       └── main.scss
├── html_templates
│   ├── index.html
│   └── page1.html
├── build
│   ├── bundle.js
│   └── index.html
├── package.json 
└── webpack.config.js
```
