# webpack scaffolding

基于webpack的脚手架, 前端框架未定

## Start 

> npm install

develop:
> npm start

build
> npm build

## Features

- es2015
- scss

## 目前已经有的demo

- regularjs 请切换到regularjs分支查看

## 目录结构说明

```js
├── app                       项目根目录
│   ├── index.js              程序入口文件
│   └── scss                  css文件夹
│       └── main.scss
├── build                        
│   ├── bundle.js
│   └── index.html
├── package.json
└── webpack.config.js
```
