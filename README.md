# webpack scaffolding

easy, clean, none-framework scaffolding

## Environment

node: v4.6.0(official recommend)

## Features

- es2015
- scss
- eslint
- file-loader
- url-loader
- html-loader
- sourcemap
- multiple generate html-file

## Start 

> npm install

develop with live-server:

> npm start

build files to path `./build`
> webpack

## file tree

```js
├── app                       
│   ├── index.js              
│   └── scss                  
│       └── main.scss
├── build                        
│   ├── bundle.js
│   └── index.html
├── package.json
└── webpack.config.js
```
